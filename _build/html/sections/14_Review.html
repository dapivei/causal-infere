14.2. Selected Problems#

14.2.1. Problem 1 (PS3 Q3)#

If we have a linear causal model, and treatment is independent of other causes (as in the case of a perfect experiment), then linear regression will estimate the linear, causal effect of $S$ on $Y$.

However, now let us suppose that a medical researcher does an experiment assigning different dosage levels of a drug, $D$, to see how dosage affects pain level $P$. The drug can have many dosage levels, so we think of it as a continuous variable. The true causal model, in real life, is

$$ P(D, U) = \gamma_1 D + \gamma_2 D^2 + U $$

That is, the drug’s effect has a linear component, $\gamma_1 D$; a non-linear component, $\gamma_2 D^2$; and an idiosyncratic component $U$.

The researcher naively runs a typical linear regression of the form $$ P = \alpha_0 + \alpha_1 D + U $$ Because it is an experiment, we can assume $U \perp D$. Thus, the researcher’s regression model will identify: $$ \alpha_1 = \frac{\text{Cov}(P, D)}{\text{Var}(D)} $$

Show, by plugging the true causal model into the above expression and using properties of the covariance, that the researcher’s regression model will estimate $\gamma_1$ plus something else.

SOLUTION:

$$

\[\begin{align*} \alpha_1 &= \frac{\text{Cov}(P, D)}{\text{Var}(D)} \\ &= \frac{\text{Cov}(\gamma_1 D + \gamma_2 D^2 + U, D)}{\text{Var}(D)} \\ &= \frac{\text{Cov}(\gamma_1 D, D) + \text{Cov}(\gamma_2 D^2, D) + \text{Cov}(U, D)}{\text{Var}(D)} \quad \text{(Prop of cov.)} \\ &= \frac{\gamma_1 \text{Cov}(D, D) + \gamma_2 \text{Cov}(D^2, D) + 0}{\text{Var}(D)} \quad \text{(Prop of cov. + $U \perp D$)} \\ &= \frac{\gamma_1 \text{Var}(D) + \gamma_2 \text{Cov}(D^2, D)}{\text{Var}(D)} \\ &= \gamma_1 + \gamma_2 \frac{\text{Cov}(D^2, D)}{\text{Var}(D)} \end{align*}\]

$$

This tells us even if we can identify the causal parameter, we can still have estimation errors from model misspecification.

See another example PS4 Q2.

14.2.2. Problem 2 (Practice Midterm Q6)#

Show, working from the definition of variance and using properties of expectations, that $$ \text{Var}(X + Y) = \text{Var}(X) + \text{Var}(Y) + 2\text{Cov}(X, Y) $$ Justify all steps (excluding simple algebra).

\[\begin{align*} \text{Var}(X + Y) &= \mathbb{E}[(X + Y - \mathbb{E}[X + Y])^2] \\ &= \mathbb{E}[(X + Y - \mathbb{E}[X] - \mathbb{E}[Y])^2] \quad \text{(prop. of expec.)} \\ &= \mathbb{E}[((X - \mathbb{E}[X]) + (Y - \mathbb{E}[Y]))^2] \\ &= \mathbb{E}[(X - \mathbb{E}[X])^2 + (Y - \mathbb{E}[Y])^2 + 2(X - \mathbb{E}[X])(Y - \mathbb{E}[Y])] \\ &= \mathbb{E}[(X - \mathbb{E}[X])^2] + \mathbb{E}[(Y - \mathbb{E}[Y])^2] + 2\mathbb{E}[(X - \mathbb{E}[X])(Y - \mathbb{E}[Y])] \quad \text{(prop. of expec.)} \\ &= \text{Var}(X) + \text{Var}(Y) + 2\text{Cov}(X, Y) \end{align*}\]

14.2.2.1. Problem 2.1 (Midterm Q6)#

Show, working from the definition of variance and using properties of expectations, that $$ \text{Var}(a + bX) = b^2 \text{Var}(X) $$ Justify all steps (excluding simple algebra). (Hint: use the first given definition of variance).

\[\begin{align*} \text{Var}(a + bX) &= \mathbb{E}[(a + bX - \mathbb{E}[a + bX])^2] \\ &= \mathbb{E}[(a + bX - \mathbb{E}[a] - b\mathbb{E}[X])^2] \quad \text{(prop. of exp.)} \\ &= \mathbb{E}[(a + bX - a - b\mathbb{E}[X])^2] \quad \text{(prop. of exp.)} \\ &= \mathbb{E}[b^2 (X - \mathbb{E}[X])^2] \\ &= b^2 \mathbb{E}[(X - \mathbb{E}[X])^2] \quad \text{(prop. of exp.)} \\ &= b^2 \text{Var}(X) \end{align*}\]

14.2.3. Problem 3 (PS4 Q3)#

How do we identify the ATE using the matching estimator under the conditional independence assumption?

Suppose we have a single control variable, $C$, that takes on $k$ values, $c_1, \dots, c_k$. Show, using the LIE, that $$ \text{ATE} = \mathbb{E}[Y(S = 1, U) - Y(S = 0, U)] = \sum_{i=1}^k \mathbb{E}[Y(S = 1, U) - Y(S = 0, U) \mid C = c_i]P(C = c_i) $$

This result follows from the LIE and the definition of the expectation of a random variable:

\[\begin{align*} \text{ATE} &= \mathbb{E}[Y(S = 1, U) - Y(S = 0, U)] \\ &= \mathbb{E}[\mathbb{E}[Y(S = 1, U) - Y(S = 0, U) \mid C]] \quad \text{(LIE)} \\ &= \sum_{i=1}^k \mathbb{E}[Y(S = 1, U) - Y(S = 0, U) \mid C = c_i]P(C = c_i) \end{align*}\]

Now, assuming that $S \perp U \mid C$, show that $$ \text{ATE} = \sum_{i=1}^k \mathbb{E}[Y(S = 1, U) - Y(S = 0, U) \mid C = c_i]P(C = c_i) $$ $$ = \sum_{i=1}^k \left(\mathbb{E}[Y(S = 1, U) \mid C = c_i, S = 1] - \mathbb{E}[Y(S = 0, U) \mid C = c_i, S = 0]\right)P(C = c_i) $$

Every term in the second line is estimable from data, implying that the ATE is identified. Indeed, this is the parameter equivalent of $\widehat{\text{ATE}}_m$ from the lecture slides.

This part is actually quite similar to showing that an experiment identifies the ATE. Basically, we’re saying that $S$ is as-good-as-random within each $C = c_i$ group, so we can treat $S$ as being like an experimentally assigned variable within groups:

\[\begin{align*} \text{ATE} &= \sum_{i=1}^k \mathbb{E}[Y(S = 1, U) - Y(S = 0, U) \mid C = c_i]P(C = c_i) \\ &= \sum_{i=1}^k \left(\mathbb{E}[Y(S = 1, U) \mid C = c_i] - \mathbb{E}[Y(S = 0, U) \mid C = c_i]\right)P(C = c_i) \quad \text{(Prop. of exp.)} \\ &= \sum_{i=1}^k \left(\mathbb{E}[Y(S = 1, U) \mid C = c_i, S = 1] - \mathbb{E}[Y(S = 0, U) \mid C = c_i, S = 0]\right)P(C = c_i) \quad (S \perp U \mid C) \end{align*}\]

14.2.4. Problem 4 (Midterm Bonus)#

Show that independence implies zero covariance. (not vice versa)

We have shown this in lectures before.

Let $X$ and $Y$ be independent random variables. By definition, the covariance of $X$ and $Y$ is given by: $$ \text{Cov}(X, Y) = \mathbb{E}[XY] - \mathbb{E}[X]\mathbb{E}[Y]. $$

Since $X$ and $Y$ are independent, the joint expectation of $XY$ can be written as the product of their marginal expectations:

\[\begin{align*} \mathbb{E}[XY] &= \mathbb{E}[\mathbb{E}[XY \mid X]] \quad \text{(LIE)} \\ &= \mathbb{E}[X \mathbb{E}[Y \mid X]] \quad \text{(property of conditional expectation)} \\ &= \mathbb{E}\left[X \int_{-\infty}^\infty y f(y \mid x) \, dy \right] \quad \text{(definition of conditional expectation)} \\ &= \mathbb{E}\left[X \int_{-\infty}^\infty y f(y) \, dy \right] \quad \text{(independence: \( Y \perp X \))} \\ &= \mathbb{E}[X \mathbb{E}[Y]] \\ &= \mathbb{E}[X]\mathbb{E}[Y]. \end{align*}\]

(This also showed that $\mathbb{E}[Y|X] = \mathbb{E}[Y]$ given independence.)

Substituting this into the covariance formula:

\[\begin{align*} \text{Cov}(X, Y) &= \mathbb{E}[XY] - \mathbb{E}[X]\mathbb{E}[Y] \\ &= \mathbb{E}[X]\mathbb{E}[Y] - \mathbb{E}[X]\mathbb{E}[Y] \\ &= 0. \end{align*}\]

Thus, if $X$ and $Y$ are independent random variables, their covariance is zero.

14.3.1. Proof 1#

Show DID estimator identifies the ATT under the parallel trends assumption.

Recall the DID estimator as the following,

$$ DID = \mathbb{E}[Y_{1,1}(S = 1, U)] - \mathbb{E}[Y_{1,0}(S = 0, U)] - \Big(\mathbb{E}[Y_{0,1}(S = 0, U)] - \mathbb{E}[Y_{0,0}(S = 0, U)]\Big) $$

and the parrlel trends assumption

$$ \mathbb{E}[Y_{1,1}(S = 0, U) - Y_{1,0}(S = 0, U)] = \mathbb{E}[Y_{0,1}(S = 0, U) - Y_{0,0}(S = 0, U)] $$

We would like to show that the DID estimator identifies ATT:

$$

\[\begin{align*} DID &= \mathbb{E}[Y_{1,1}(S = 1, U)] - \mathbb{E}[Y_{1,0}(S = 0, U)] - (\mathbb{E}[Y_{0,1}(S = 0, U)] - \mathbb{E}[Y_{0,0}(S = 0, U)]) \\ &= \mathbb{E}[Y_{1,1}(S = 1, U)] - \mathbb{E}[Y_{1,1}(S = 0, U)] + \mathbb{E}[Y_{1,1}(S = 0, U)] - \mathbb{E}[Y_{1,0}(S = 0, U)] - (\mathbb{E}[Y_{0,1}(S = 0, U)] - \mathbb{E}[Y_{0,0}(S = 0, U)]) \\ & = ATT + \mathbb{E}[Y_{1,1}(S = 0, U)] - \mathbb{E}[Y_{1,0}(S = 0, U)] - (\mathbb{E}[Y_{0,1}(S = 0, U)] - \mathbb{E}[Y_{0,0}(S = 0, U)]) \\ &= ATT + 0 \quad \text{(Parr. trends)} \end{align*}\]

$$

14.3.3. Proof 3#

How did we identify the RDD estimator with the smoothness assumption?

\[\begin{align*} \mathbb{E}[Y \mid R = c] &= \mathbb{E}[Y(S = 1, U) \mid R = c] \quad \text{(r $\geq$ c $\implies$ S = 1)} \\ \lim_{r \uparrow c} \mathbb{E}[Y \mid R = r] &= \lim_{r \uparrow c} \mathbb{E}[Y(S = 0, U) \mid R = r] \quad \text{(r $<$ c $\implies$ S = 0)} \\ &= \mathbb{E}[Y(S = 0, U) \mid R = c] \quad \text{(smoothness)}. \end{align*}\]
\[\begin{align*} \implies \mathbb{E}[Y(S = 1, U) - Y(S = 0, U) \mid R = c] &= \mathbb{E}[Y(S = 1, U) \mid R = c] - \lim_{r \uparrow c} \mathbb{E}[Y(S = 0, U) \mid R = r]. \end{align*}\]

We can estimate this using local linear regression below. Why does $\alpha_1$ estimate the above effect, and what does each parameter do here?

$$ Y = \alpha_0 + \alpha_1 S + \alpha_2 (R - c) + \alpha_3 S \cdot (R - c) $$

14.3.4. Proof 4#

Show that the Wald estimator identifies LATE.

The Wald estimator can be written as the following:

\[\begin{align*} \alpha_1^{IV} &= \frac{\text{Cov}(Z, Y)}{\text{Cov}(Z, S)} = \frac{\text{Cov}(Z, Y) / \text{Var}(Z)}{\text{Cov}(Z, S) / \text{Var}(Z)} = \frac{\mathbb{E}[Y \mid Z = 1] - \mathbb{E}[Y \mid Z = 0]}{\mathbb{E}[S \mid Z = 1] - \mathbb{E}[S \mid Z = 0]}. \end{align*}\]

We expand the numerator and denominator separately.

\[\begin{align*} \mathbb{E}[Y \mid Z = 1] - \mathbb{E}[Y \mid Z = 0] &= \mathbb{E}[Y(S, U) \mid Z = 1] - \mathbb{E}[Y(S, U) \mid Z = 0] \\ &= \mathbb{E}[Y(S(Z = 1, V), U) \mid Z = 1] - \mathbb{E}[Y(S(Z = 0, V), U) \mid Z = 0] \\ &= \mathbb{E}[Y(S(Z = 1, V), U)] - \mathbb{E}[Y(S(Z = 0, V), U)] \quad \text{(Instr. Exog.)} \\ &= \mathbb{E}[Y(S(Z = 1, V), U) - Y(S(Z = 0, V), U)] \\ &= \mathbb{E}[\mathbb{E}[Y(S(1, V), U) - Y(S(0, V), U) \mid S(1, V), S(0, V)]] \quad \text{(LIE)} \\ &= \mathbb{E}[Y(S = 1, U) - Y(S = 0, U) \mid \text{Complier}]P(\text{Complier}) \\ &\quad + \mathbb{E}[Y(S = 1, U) - Y(S = 0, U) \mid \text{Defier}]P(\text{Defier}) \\ &= \mathbb{E}[Y(S = 1, U) - Y(S = 0, U) \mid \text{Complier}]P(\text{Complier}) \quad \text{(Monot.)}. \end{align*}\]

Now for the denominator,

\[\begin{align*} \mathbb{E}[S \mid Z = 1] - \mathbb{E}[S \mid Z = 0] &= \mathbb{E}[S(Z, V) \mid Z = 1] - \mathbb{E}[S(Z, V) \mid Z = 0] \\ &= \mathbb{E}[S(Z = 1, V) \mid Z = 1] - \mathbb{E}[S(Z = 0, V) \mid Z = 0] \\ &= \mathbb{E}[S(Z = 1, V)] - \mathbb{E}[S(Z = 0, V)] \quad \text{(Instr. Exog.)} \\ &= \mathbb{E}[S(Z = 1, V) - S(Z = 0, V)] \\ &= \mathbb{E}[\mathbb{E}[S(Z = 1, V) - S(Z = 0, V) \mid S(1, V), S(0, V)]] \quad \text{(LIE)} \\ &= \mathbb{E}[1 - 0 \mid \text{Complier}]P(\text{Complier}) + \mathbb{E}[0 - 1 \mid \text{Defier}]P(\text{Defier}) \\ &= P(\text{Complier}) \quad \text{(Monot.)}. \end{align*}\]

Putting it all together,

\[\begin{align*} \alpha_1^{IV} &= \frac{\mathbb{E}[Y \mid Z = 1] - \mathbb{E}[Y \mid Z = 0]}{\mathbb{E}[S \mid Z = 1] - \mathbb{E}[S \mid Z = 0]} \\ &= \frac{\mathbb{E}[Y(S = 1, U) - Y(S = 0, U) \mid \text{Complier}]P(\text{Complier})}{P(\text{Complier})} \\ &= \mathbb{E}[Y(S = 1, U) - Y(S = 0, U) \mid \text{Complier}] \quad \text{(Instr. Rel.)} \\ &= \text{LATE}. \end{align*}\]

14.3.5. Proof 5#

Show the common alternative form of the covariance definition: $$ \text{Cov}(X, Y) = \mathbb{E}[XY] - \mathbb{E}[X]\mathbb{E}[Y] $$

\[\begin{align*} \text{Cov}(X, Y) &= \mathbb{E}[(X - \mathbb{E}[X])(Y - \mathbb{E}[Y])] \\ &= \mathbb{E}[XY - X\mathbb{E}[Y] - Y\mathbb{E}[X] + \mathbb{E}[X]\mathbb{E}[Y]] \\ &= \mathbb{E}[XY] - \mathbb{E}[X]\mathbb{E}[Y] - \mathbb{E}[Y]\mathbb{E}[X] + \mathbb{E}[X]\mathbb{E}[Y] \\ &= \mathbb{E}[XY] - 2\mathbb{E}[X]\mathbb{E}[Y] + \mathbb{E}[X]\mathbb{E}[Y] \\ &= \mathbb{E}[XY] - \mathbb{E}[X]\mathbb{E}[Y] \end{align*}\]
